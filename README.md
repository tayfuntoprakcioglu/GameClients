# Game Clients

**The Ultimate Gaming Shell Re-imagined.**

Game Clients is a custom shell replacement for Windows 10 & 11, specifically optimized for modified OS versions like **AtlasOS**. It replaces the standard Windows Desktop (Explorer) with a lightweight, controller-friendly gaming interface, designed to provide the cleanest, fastest gaming experience possible.

## Installation

1.  **Download:** Get the latest `GameClients_Beta.zip` and extract the contents to `C:\Games\GameClients`.
2.  **Run Setup:** Right-click `GameClients_START.cmd` and select **Run as Administrator**.
3.  **Install:** Select option `1` in the command window to configure the Game Clients Shell.
4.  **Reboot:** Restart your PC. You will automatically boot into the Game Clients interface instead of the standard Windows Desktop.

> **Note:** To revert to the standard Windows Desktop, use the "Uninstall" or "Restore" option provided in the setup script or via the dashboard settings.

## Compatibility

*   **Optimized for:** [Atlas OS](https://atlasos.net/) (Highly Recommended for best performance)
